## Folder Structure

| File / Folder           | Purpose                                     |
| ----------------------- | ------------------------------------------- |
| `app/`                  | UI5 app module                              |
| `approuter`             | Acts as a proxy to service from UI          |
| `srv/`                  | All service related files                   |
| `db/`                   | Database module                             |
| `package.json`          | project metadata and configuration          |
| `tests/api-integration` | contains API tests                          |
